# SpaceGlowTD

SpaceGlowTD to prosta gra 3D typu Tower Defense stworzona w C++ i OpenGL z mapą generowaną proceduralnie. Projekt demonstruje podstawy grafiki komputerowej poprzez implementację shaderów, oświetlenia oraz logiki walki.

## Instrukcja uruchomienia

1. Stwórz folder budowania: `mkdir build && cd build`
2. Wygeneruj pliki projektu: `cmake ..`
3. Skompiluj projekt: `cmake --build .`
4. Uruchom wygenerowany plik wykonywalny wewnątrz folderu `build`.
